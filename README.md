This restful API was built with Express, MongoDB, and Mongoose.
It supports creating, reading , updating and deleting(CRUD).
API Routes: 
Create New Book: POST http://localhost:4000/api/books/add
Get All Books: GET http://localhost:4000/api/books
Update Book: PUT http://localhost:4000/api/books/:id
Delete Book: DELETE http://localhost:4000/api/books/delete/:id
